# Nexus Truth Constitution

Nexus must be judged only on proven behavior, not structural presence.

## Gate A — AI OS truth
- continuity survives restart
- memory changes behavior
- state resolves to action
- execution persists and recovers
- approvals block and audit real actions
- UI reflects live operational truth

## Gate B — Autonomous product company
- opportunity discovery and ranking
- product generation pipeline
- packaging and distribution
- economics and feedback loops
- fleet maintenance and iteration
- policy-bounded autonomy

## Rule
If it is not proven in runtime + evidence + validator, it is not done.
